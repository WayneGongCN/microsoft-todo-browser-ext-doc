Microsoft To Do 浏览器扩展是一款免费的开源软件。

- 支持通过扩展弹窗快速创建 Todo 任务；
- 支持自动填充选中内容到任务标题/描述；
- 支持添加 Todo 任务时保存当前页面的书签信息；
